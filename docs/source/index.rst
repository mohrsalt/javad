JaVAD: Just Another Voice Activity Detector
===========================================

Installation
============

You can install the package using pip:

.. code-block:: bash

    pip install javad

To install JaVAD with additional features: audio loading, quick processing, use following command:

.. code-block:: bash

    pip install javad[extras]

Requirements
============

- Python 3.8+
- PyTorch 2.0.0+
- NumPy 1.20.0+
- Optional: `soundfile` for additional audio processing features
